# RickRoll Link

## Create Rick Roll Links Easily! With A Docker Container!
# Installation
Run This Command To Install:

```
docker pull yasir6/rickroll-link
```
Run This Command To Run The Container:
```
docker run -p <host_port>:8000 yasir6/rickroll-link
```
